# 🚀 Links Úteis para Tecnologia e Programação

Bem-vindo ao hub colaborativo de links úteis sobre tecnologia da informação e programação! Aqui você encontra recomendações organizadas por áreas de conhecimento, reunindo:

- Artigos e tutoriais
- Vídeos e palestras
- Livros
- Cursos
- Ferramentas e documentação
- Podcasts, newsletters e comunidades

Cada arquivo `.md` representa uma área específica, facilitando a busca por recursos relevantes.

## Como contribuir

- Siga o template disponível em `.github/TEMPLATE.md` para adicionar novas recomendações.
- Inclua sempre o título, link, breve descrição, tags, nível, indicação de gratuito/pago e data.
- Prefira links oficiais ou de fontes reconhecidas.
- Evite duplicidade de recomendações.

## Exemplo de recomendação

```
### [#](https://exemplo.com/css) Curso de CSS Moderno

**Tags:** [CSS, Front-end, Curso]
**Nível:** Iniciante
**Gratuito:** Sim
**Data:** 2026-03-15

**Previa:**
> _Curso completo sobre CSS, abordando layouts responsivos, flexbox e grid._
```

## Organização

- Cada área possui um arquivo `.md` próprio (ex: `css.md`, `javascript.md`, `inteligencia-artificial.md`).
- Recomendações são agrupadas por categoria (artigos, vídeos, livros, etc).
- O arquivo `AGENT.md` orienta agentes de IA sobre como criar e preencher arquivos.
