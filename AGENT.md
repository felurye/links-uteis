# Guia para Agentes de IA

Este repositório é um hub de links úteis relacionados à tecnologia da informação e programação. Cada área de conhecimento é separada por um arquivo `.md` que pode conter indicações de livros, cursos, palestras, tutoriais/artigos, etc.

## Orientações para criação e preenchimento de arquivos

- Utilize o template disponível em `.github/TEMPLATE.md` como base.
- Ao adicionar recomendações, inclua obrigatoriamente:
  - **Título**: Nome do recurso.
  - **Link**: URL para acesso.
  - **Brevia**: Pequena descrição do conteúdo.
- Categorize os recursos (livros, cursos, palestras, tutoriais, artigos, ferramentas, podcasts, comunidades, newsletters, etc).
- Prefira links oficiais ou de fontes reconhecidas.
- Evite duplicidade de recomendações.
- Mantenha a formatação Markdown para facilitar leitura e automação.
- Adicione tags ou palavras-chave relevantes para facilitar buscas.
- Caso crie uma nova área, siga o padrão de nomeação e estrutura do template.

## Sugestões de incrementos para os arquivos

- Adicionar seção de "Ferramentas" (softwares, plugins, extensões).
- Incluir "Podcasts" e "Newsletters".
- Seção de "Comunidades" (grupos, fóruns, Discord, Telegram).
- Tags ou palavras-chave para cada recurso.
- Indicação de nível (iniciante, intermediário, avançado).
- Sinalizar recursos gratuitos ou pagos.
- Adicionar data de inclusão/revisão.
- Sugestão de ordem de estudo/prioridade.
